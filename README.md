Number Guessing Game
A simple, fun and interactive console-based game written in Python.

The computer generates and randomly selects a number between 1 and 100.
A limited number of attempts, up to 10 is given.
After each game, points are awarded according to how fast you guess the number.

Features
1. Random number generator (1-100)
2. Limit of 10 attempts.
3. Prompted hints: "Too high!/Too Low!"
4. Validated inputs (Only Integers count)
5. Scoring system and points awarded based off of attempts
6. Stores and reveals your guesses made in every attempt.

Scoring system:

  ≤ 3 attempts → 5 points
  
  ≤ 7 attempts → 3 points
  
  ≥ 7 attempts → 1 point

  10 attempts = 0 point

How to Play
1. Run the Game
2. Guess the number when prompted
3. Follow given hints: "Too High! / Too Low!"
4. Win by guessing the number or use all 10 attempts → see your guesses
5. Choice of whether to play again
6. Decide to quit, and see your games played and total score.

Personal Finance Simulator with Investment Projections

A simulator that simulates real-world budgeting, expense tracking, and compound interest-based saving projections mirroring features found in entry-level fintech tools.

Allows user input to select and add transactions where they're sorted out into income and expense. 
Code calculates the net value they'll be left with.
Allows user to view a summary of all transactions that occurred.
Allows user to project investmemnts for 1, 3 or 5 years with the amount they decide on.

Features
1. Add transactions of various types
2. Transactions are classified accordingly
3. Generates a summarized version of all transactions.
4. Calculates net value for user.
5. Projects investments for 1, 3 or 5 years compounded anually.
6. Re-prompts users if there are invalid inputs.
